Python Base

Corso Python Base

Fondamenti del linguaggio Python, strutture dati, file, funzioni, API REST e gestione JSON per interagire con servizi esterni in modo consapevole.

Python Script, sintassi, stringhe e ambiente di sviluppo.
Dati Liste, tuple, dizionari, set, file, NumPy e Pandas.
API REST Endpoint, request, response e metodi HTTP.
JSON Parsing, accesso ai dati, errori e buone pratiche.

Moduli

Percorso progressivo per partecipanti con conoscenze informatiche di base.

  • M1 Fondamenti di Python Installazione, esecuzione, prime istruzioni, sintassi, tipi numerici e stringhe.
  • M2 Strutture dati, logica e file Container, funzioni, moduli, eccezioni, NumPy e Pandas per l'analisi dati.
  • M3 API, JSON e integrazione con Python Concetti REST, metodi HTTP, libreria requests, parsing JSON, errori e sicurezza.

Obiettivo del corso

Fornire le basi del linguaggio Python e introdurre, a livello teorico e pratico, il funzionamento delle API REST e la gestione di dati JSON con Python.

Competenze in uscita

  • Scrivere script Python di base.
  • Comprendere e usare le principali strutture dati.
  • Leggere e scrivere file.
  • Effettuare chiamate API e gestire JSON.
